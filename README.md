## Motocross ASMR App – Privacy Policy

This web application allows the user to upload and publish videos to TikTok using the official TikTok API.
We collect and store:
TikTok OAuth access tokens (securely stored)
Uploaded video files for publishing purposes
We do not:
Sell user data
Share data with third parties
Collect unnecessary personal information
Data is used solely to publish videos to the authenticated TikTok account.
